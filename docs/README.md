# GemmaFC Docs

This folder stores durable project decisions and setup notes for GemmaFC.

## Files

- `design-decisions.md`: Living decision log for product, architecture, benchmark, and demo choices.
- `google-research-football-setup.md`: Local Docker setup and smoke-test commands for Google Research Football.
- `app-ui.md`: Docker app UI build/run notes.

## How To Update

Add dated entries when a choice affects implementation, demo framing, benchmark fairness, or judging clarity. Prefer short, concrete rationale over long prose.
